Проверка:
ps aux | grep nginx
(Ты должен увидеть, что worker-процессы NGINX запущены от www-data.)

Переключение между пользователями:
su - appuser



Перезапусти NGINX:
nginx -s reload
nginx -t && nginx -s reload



cat /usr/local/bin/start-hls.sh



ПРОВЕРКА:
tail -f /var/log/nginx/error_rtmp.log | grep --line-buffered -E '(publish|play|createStream|client connected)'



ПРАВА НА ПАПКИ ЗАПИСИ:

mkdir -p /srv/streaming/{live,archive}
chown -R www-data:www-data /srv/streaming
chmod -R 775 /srv/streaming

ПРОВЕРКА ПРАВ КАТАЛОГА:
ls -ld /srv/streaming /srv/streaming/live


ПЕЕРЕЗАПУСК
systemctl daemon-reload
systemctl reset-failed
