## 🔧 Ключевые изменения

### ❌ Было (проблемы):
```typescript
// exec() - буферизует ВСЕ логи в память
const { stdout, stderr } = await execAsync(ffmpegCommand, {
  maxBuffer: 100 * 1024 * 1024  // 100MB буфер
})
```

### ✅ Стало (решение):
```typescript
// spawn() - НЕ буферизует
const ffmpeg = spawn('ffmpeg', args)

ffmpeg.stderr.on('data', (data) => {
  // Просто логируем, НЕ накапливаем
  console.log(data.toString())
})
```

               │
               ▼
     ┌──────────────────┐

     │  Статусы задач   │
     └──────┬───────────┘
            │
            ▼
┌──────────────────────────────────────────────┐
│       video-worker (6GB RAM)                 │
│  - concurrency: 1 (один за раз)              │
│  - spawn(ffmpeg) БЕЗ буферизации             │
│  - try/finally → всегда чистит temp          │

└──────────────────────────────────────────────┘

3. Проверь что ffmpeg не падает сам: `ffmpeg -version`
4. Уменьши битрейт сжатия (вместо `-crf 26` → `-crf 30`)
