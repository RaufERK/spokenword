## 🚀 КАК ПОЛЬЗОВАТЬСЯ СИСТЕМОЙ

### Для АДМИНА:

#### 1. Создание пакета материалов
- Перейти на `/admin/packages`
- Нажать "Создать пакет"
- Заполнить название, описание, цену
- Нажать "Создать пакет"

#### 2. Загрузка лекций
- После создания пакета откроется страница управления лекциями
- Нажать "Выбрать аудио файлы"
- Выбрать несколько аудио файлов (MP3, WAV, M4A, FLAC)
- Файлы автоматически сжимаются с 3GB до ~200MB
- Прогресс отображается в реальном времени

#### 3. Управление доступом пользователей
- Перейти на `/users`
- Найти нужного пользователя
- Нажать кнопку "Управлять" в колонке "Материалы"
- В модальном окне поставить галочки напротив купленных пакетов
- Добавить заметку о способе оплаты
- Сохранить

### Для ПОЛЬЗОВАТЕЛЕЙ:

#### 1. Просмотр материалов
- Перейти на `/paid-content`
- Увидеть свои купленные материалы (зеленые блоки)
- Увидеть доступные для покупки (синие блоки)

#### 2. Прослушивание лекций
- Развернуть купленный пакет
- Нажать кнопку воспроизведения у любой лекции
- Использовать встроенный плеер с защитой от скачивания

## 🏗️ АРХИТЕКТУРА СИСТЕМЫ

### База данных:
```
ContentPackage (пакеты)
├── id, title, description, price
├── isActive, createdAt, uploadedBy
└── связи: items[], purchases[]

PackageItem (лекции)
├── id, packageId, title, fileName
├── filePath, duration, orderIndex
├── originalSize, compressedSize
└── связь: package

UserPackageAccess (покупки)
├── id, userId, packageId
├── purchaseDate, price, grantedBy
├── notes (способ оплаты)
└── связи: user, package, admin
```

### Файловая структура:
```
paid-content/
├── packages/
│   ├── package_1/
│   │   ├── lecture_01_compressed.mp3
│   │   ├── lecture_02_compressed.mp3
│   │   └── ...
│   └── package_2/
│       └── ...
└── temp/ (временные файлы)
```

### API маршруты:
```
/api/admin/packages/
├── GET, POST - управление пакетами
├── /upload - загрузка и сжатие аудио
└── /[id]/items - управление лекциями

/api/admin/users/
├── /[id]/packages - получение доступов пользователя
└── /package-access - предоставление/отзыв доступа

/api/paid-content/packages/[packageId]/items/[itemId]/stream
└── защищенный стриминг аудио
```

## 📈 АВТОМАТИЧЕСКОЕ СЖАТИЕ

### FFmpeg настройки:
```bash
ffmpeg -i input.wav \
  -codec:a mp3 \
  -b:a 64k \        # Битрейт для речи
  -ar 22050 \       # Частота дискретизации
  -ac 1 \           # Моно
  output.mp3
```

### Результат сжатия:
- **Было:** 3GB WAV файл
- **Стало:** ~200MB MP3 файл
- **Сжатие:** 15x меньше размер
- **Качество:** Отлично для речи/лекций
