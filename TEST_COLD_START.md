# 🧪 Тестирование исправления холодного старта

## ✅ Что уже сделано (локально):

### 1. **API изменён** (`app/api/stream-status/route.ts`)
- Порог для `isWarmingUp`: 4 → **8 сегментов** (16 секунд)
- Стрим не будет показываться как "live" пока не прогреется

### 2. **HLS.js клиент улучшен** (`components/HlsPlayer.tsx`)
- `maxBufferLength`: 20 → **30 секунд**
- `maxMaxBufferLength`: 40 → **60 секунд**
- `liveSyncDuration`: 3 → **5 секунд**
- `liveMaxLatencyDuration`: 10 → **15 секунд**

### 3. **UI добавлен экран "прогрева"** (`app/live/page.tsx`)
- Теперь есть 3 состояния:
  - ⚪ Стрим offline
  - 🔵 Стрим прогревается (новое!)
  - 🔴 Стрим в эфире
- Проверка статуса: каждые 10 сек → **5 секунд**
