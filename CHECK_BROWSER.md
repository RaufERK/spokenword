### 1. Открой DevTools (F12):
```
1. Открой: https://spokenword.ru/admin/packages
2. Нажми F12 (DevTools)
3. Вкладка "Network" (Сеть)
4. Попробуй загрузить файл
```

### 2. Смотри что происходит:
```
Есть ли запрос: POST /api/admin/packages/upload ?

Если ДА:
  - Какой статус? 200? 413? 500?
  - Что в Response?
  - Что в Request (size)?

Если НЕТ:
  - Ошибка в Console?
  - Кнопка disabled?
  - JavaScript ошибка?
```

### 3. Вкладка "Console":
```
Есть ли ошибки JavaScript?
Красный текст?
```

## 🐛 ВОЗМОЖНЫЕ ПРИЧИНЫ:

### 1. **Build не обновился**
Старая версия JS в браузере (кэш)

**Решение:** Ctrl+Shift+R (hard refresh)

### 2. **JavaScript ошибка**
Код сломался при билде

**Решение:** Смотри Console

### 3. **Кнопка disabled**
Что-то блокирует загрузку

**Решение:** Проверь UI состояние

### 4. **CORS/Auth проблема**
Сессия истекла

**Решение:** Перелогинься
