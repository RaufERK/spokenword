## 🎯 ЧТО НУЖНО СДЕЛАТЬ ТЕБЕ:

### ВАРИАНТ 1: Зайди сам и создай swap (5 минут)
```bash
ssh amster_app

# Введи пароль sudo и выполни:
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
sudo swapon --show

# Сделай постоянным:
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

# Проверь:
free -h

# Выйди:
exit
```

### ВАРИАНТ 2: Настрой sudo без пароля (для автоматизации)
```bash
ssh amster_app
sudo visudo

# Добавь строку:
appuser ALL=(ALL) NOPASSWD: /usr/bin/fallocate, /usr/bin/chmod, /usr/bin/mkswap, /usr/bin/swapon, /usr/bin/tee

# Сохрани и выйди
```
