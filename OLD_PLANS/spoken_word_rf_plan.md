## ⚙️ ЭТАП 1 — НАСТРОЙКА СЕРВЕРА

### 1. Доступ
```bash
ssh root@IP
apt update && apt upgrade -y
```

### 2. Безопасность
```bash
adduser appuser
usermod -aG sudo appuser
```

- настроить SSH по ключу  
- отключить пароль  

### 3. SSH между серверами
- РФ → Амстердам  
- Амстердам → РФ  

Проверка:
```bash
ssh amster_app
scp file amster_app:/path
```
