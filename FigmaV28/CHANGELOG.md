```

│☑│№ │Имя │Фамил.│Логин │Город │Телефон │Дата оплаты │Управление│Профиль │

#### Массовые операции (всегда видны сверху):
```

#### Фильтры:
- Все
- С оплатой (активная оплата < 1 месяца)
- Без оплаты
- **Новые** (у кого вообще нет оплаты)
- **Админы** (ADMIN + SUPER)

#### Особенности:
✅ Фиксированный заголовок (sticky top) - не уезжает при прокрутке
✅ 50 тестовых пользователей для демонстрации

✅ Дата оплаты и управление в отдельных колонках

### 6. 💬 Чат - ПОЛНОСТЬЮ ПЕРЕРАБОТАН

#### Новая структура (как Telegram):
```
┌────────────┬──────────────────────────┐
│ Список     │  Активный чат            │
│ чатов      │                          │
├────────────┤                          │
│📢 Общий    │ [Заголовок чата]         │
│  чат       │                          │
│            │ ┌──────────────────────┐ │
│🛟 Поддержка│ │ Сообщения            │ │
│   (2)      │ │                      │ │
│            │ │ Елеонора:            │ │
│💬 Елеонора │ │ "Спасибо!"           │ │
│            │ │ [👍 5] [❤️ 3]         │ │
│💬 Алексей  │ │                      │ │
│            │ └──────────────────────┘ │
│            │                          │
│            │ [Поле ввода...]          │
│            │ [Отправить]              │
└────────────┴──────────────────────────┘
```

#### Типы чатов:

**📢 Общий чат:**
- Виден всем пользователям
- Клик на имя → открывает личный чат

**🛟 Поддержка:**
- Обращения к администрации
- Пользователи видят свои обращения
- Админы видят ВСЕ обращения

**💬 Личные чаты:**
- Создаются при клике на имя в общем чате
- Приватная переписка 1-на-1
- Автоматически добавляются в список

#### Функции:
✅ Счётчики непрочитанных (красный бейдж)
✅ Последнее сообщение в списке чатов
✅ Переключение между чатами
✅ Реакции на сообщения (👍❤️😊🔥👏)
✅ Ссылки в сообщениях
✅ Удаление сообщений (для админов)
✅ Бейджи ролей (ADMIN, SUPER)

## 📊 Массовые операции пользователей

### Дать доступ (зелёная кнопка):
1. Выбрать пользователей галочками
2. Нажать "Дать доступ"
3. Выбрать мероприятие
4. Подтвердить → всем выбранным активируется оплата

### Отозвать доступ (красная кнопка):
1. Выбрать пользователей
2. Нажать "Отозвать доступ"
3. Подтвердить → у всех отзывается оплата

### Платный контент (фиолетовая кнопка):

## ⚡ Как использовать

### Таблица пользователей:
1. Открыть **Админка → Пользователи**
2. Выбрать пользователей галочками
3. Использовать массовые операции
4. Прокручивать таблицу - заголовок остаётся сверху

### Мероприятия:
1. Открыть **Админка → Мероприятия**
2. Нажать "+ Создать мероприятие"
3. Заполнить форму с датапикером
4. Мероприятие появится в списках оплаты и загрузки

### Чат:
1. Открыть **Чат** в меню
2. Выбрать чат из левой панели (Общий / Поддержка)
3. В общем чате кликнуть на имя → создастся личный чат
4. Переписка сохраняется, счётчики непрочитанных работают
