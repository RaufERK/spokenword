## 🔍 ПРОБЛЕМА:

**Next.js загружает ВЕСЬ файл в память перед отправкой в очередь!**

```typescript
// app/api/admin/packages/upload/route.ts
const bytes = await file.arrayBuffer()  ← ЗАГРУЖАЕТ ВСЁ В RAM!
const buffer = Buffer.from(bytes)
await writeFile(tempFilePath, buffer)
```

**Для файла 509MB:**
- Загрузка в RAM: 509MB
- Buffer: 509MB
- Итого: 1GB+ памяти
- Next.js лимит: 2GB
- **Результат:** Крашится при 2-3 больших файлах одновременно!

## 🎯 РЕШЕНИЕ ПРОБЛЕМЫ (на будущее):

### Использовать стриминг вместо arrayBuffer:

```typescript
// Вместо:
const bytes = await file.arrayBuffer()
const buffer = Buffer.from(bytes)
await writeFile(tempFilePath, buffer)

// Делать:
const stream = file.stream()
const writeStream = createWriteStream(tempFilePath)
await stream.pipeTo(Writable.toWeb(writeStream))
```

**Преимущество:**
- Не загружает весь файл в память
- Пишет файл кусками (chunks)
- Next.js не крашится
