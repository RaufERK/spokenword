## 🚀 Быстрый старт

### 1. Прочитай

```bash
open _ICECAST_SETUP/START_HERE.txt
```

### 2. Загрузи на сервер

```bash
cd _ICECAST_SETUP
scp -r * amster:/root/icecast-setup/
```

### 3. Установи

```bash
ssh amster
cd /root/icecast-setup
sudo ./install-icecast.sh
```

### 4. Следуй инструкциям

Обновить nginx → Деплой frontend → Тест

**Время:** 15-30 минут
