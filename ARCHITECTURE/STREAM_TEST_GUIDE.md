# 🧪 Краткая инструкция по тестированию стрима

## Подготовка (1 минута)

```bash
# 1. Задеплоить изменения
npm run deploy

# 2. Перезапустить приложение
ssh amster 'sudo -u appuser bash -c "export PATH=/home/appuser/.nvm/versions/node/v22.18.0/bin:$PATH && pm2 restart spokenword"'
```

---

## Тестирование (5 минут)

### Шаг 1: Откройте браузер

1. Откройте **Chrome** или **Firefox**
2. Перейдите на: https://spoken-word.ru/live
3. Откройте **DevTools** (F12) → вкладка **Console**
4. Страница должна показывать "Сейчас трансляция не ведётся"

### Шаг 2: Запустите стрим

1. Откройте **OBS Studio**
2. Проверьте настройки:
   - Сервер: `rtmp://stream.spoken-word.ru/live`
   - Ключ: `main`
3. Нажмите **"Начать трансляцию"**

### Шаг 3: Наблюдайте

**Ожидаемое поведение:**

1. **Через 5-10 секунд:**
   - Экран изменится на "🎬 Стрим запускается..."
   - Увидите анимацию с двойным спиннером
   - В консоли: `🎬 Stream just started! Entering warm-up period...`

2. **Через 6 секунд:**
   - Появится видео плеер
   - В консоли: `✅ Warm-up complete, loading player...`
   - Видео начнет загружаться

3. **Через 8-10 секунд:**
   - Видео начнет воспроизводиться
   - В консоли: `HLS manifest parsed, starting playback`

### Шаг 4: Проверьте логи

В **Console** должны быть такие сообщения:

```
Stream status: {isLive: true, isWarmingUp: true, segmentCount: 2, ...}
🎬 Stream just started! Entering warm-up period...
✅ Warm-up complete, loading player...
🎬 Loading stream: {streamUrl: "...", isYoungStream: true, streamAge: 8, ...}
HLS manifest parsed, starting playback
```

## ❌ Если что-то не работает

### Проблема: Не появляется "Стрим запускается..."

**Проверьте API:**
```bash
curl "https://spoken-word.ru/api/stream-status?key=main"
```

Должно вернуть: `{"isLive": true, "isWarmingUp": true, ...}`

### Проблема: Видео не загружается после 6 секунд

**Проверьте сегменты на сервере:**
```bash
ssh amster 'ls -lah /var/lib/srs/hls/live/'
```

Должны быть файлы: `main.m3u8`, `main-0.ts`, `main-1.ts`, и т.д.

### Проблема: Ошибки в консоли

**Скопируйте все сообщения из Console и пришлите мне:**
- Сообщения с 🎬, 🔄, ✅, ❌
- Красные сообщения об ошибках
- Вывод команды: `curl "https://spoken-word.ru/api/stream-status?key=main"`

## 🔄 Повторный тест

Чтобы проверить снова:

1. **Остановите стрим** в OBS
2. **Обновите страницу** (Ctrl+Shift+R)
3. **Подождите 10 секунд** (чтобы API обнаружил offline)
4. Страница должна показать "Сейчас трансляция не ведётся"
5. **Запустите стрим** снова
6. Повторите наблюдение
