# 🎬 Решение проблем со стримингом SpokenWord

## 📋 Что было сделано

### 1. **Анализ проблемы**
- Стрим прерывался из-за неправильной структуры файлов HLS
- Файлы .ts создавались в корне `/srv/streaming/live/`, но запрашивались из `/srv/streaming/live/main/`
- Отсутствовала автоматическая синхронизация файлов
- Мобильные устройства особенно чувствительны к отсутствующим сегментам

### 2. **Реализованные решения**

#### **HLS Monitor** (`/usr/local/bin/hls-monitor.sh`)
- Автоматически создает символические ссылки для всех HLS файлов
- Следит за появлением новых сегментов
- Очищает битые ссылки
- Восстанавливает права доступа

#### **Stream Watchdog** (`/usr/local/bin/stream-watchdog.sh`)
- Запускается каждую минуту через cron
- Проверяет статус стрима
- Автоматически исправляет проблемы
- Перезапускает сервисы при критических ошибках

#### **Оптимизированная конфигурация nginx**
- `drop_idle_publisher 0` - не отключает неактивный поток
- `hls_continuous on` - непрерывная нумерация сегментов
- `hls_sync 100ms` - точная синхронизация времени
- Оптимальные размеры буферов для мобильных

#### **Stream Manager** (`scripts/stream-manager.sh`)
- Централизованное управление всей системой
- Быстрая диагностика проблем
- Простые команды для исправления

## 🚀 Как запустить стрим

### 1. **Проверка готовности системы**
```bash
npm run stream:status
```

### 2. **Настройка OBS Studio**
```bash
npm run stream:obs
```

Настройки:
- **Сервер:** `rtmp://185.200.178.73/live`
- **Ключ потока:** `main`
- **Видеокодек:** x264
- **Битрейт:** 2500-4000 Kbps
- **Keyframe Interval:** 2 секунды
- **Разрешение:** 1920x1080 или 1280x720
- **FPS:** 30

### 3. **Запуск стрима**
1. Начните трансляцию в OBS
2. Проверьте статус:
   ```bash
   npm run stream:status
   ```
3. Откройте стрим:
   - **Веб:** https://spoken-word.ru/live
   - **Мобильный:** тот же адрес

## 🔍 Диагностика проблем

### **Стрим не запускается:**
1. Проверьте настройки OBS (сервер и ключ)
2. Выполните `npm run stream:recovery`
3. Проверьте файрвол на порт 1935

### **Стрим прерывается:**
1. Проверьте стабильность интернета
2. Уменьшите битрейт в OBS
3. Выполните `npm run stream:fix`

### **Не работает на мобильных:**
1. Проверьте HTTPS сертификат
2. Убедитесь что CORS заголовки настроены
3. Попробуйте открыть прямую ссылку на m3u8
