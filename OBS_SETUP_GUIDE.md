# 🎥 Инструкция по настройке OBS Studio для потоковой трансляции

## 📋 Оптимальные настройки для текстового контента

Эта конфигурация оптимизирована для трансляции **текста (10-12 строк)** с максимальной надёжностью на мобильных устройствах.

---

## ⚙️ Настройки OBS Studio

### 1️⃣ **Настройки Вещания (Stream)**

**Файл → Настройки → Вещание**

- **Сервис:** `Пользовательский`
- **Сервер:** `rtmp://spoken-word.ru/live`
- **Ключ трансляции:** `main`

---

### 2️⃣ **Настройки Вывода (Output)**

**Файл → Настройки → Вывод**

#### Режим вывода: **Расширенный**

#### Вкладка "Streaming":

**Кодировщик видео:**
- **Кодировщик:** `x264` (программный) или `NVENC H.264` (если есть NVIDIA GPU)
- **Управление битрейтом:** `CBR` (постоянный битрейт)
- **Битрейт:** `1500 Kbps` ⭐
- **Интервал ключевых кадров:** `2` секунды ⭐⭐⭐ **ВАЖНО!**
- **Профиль:** `baseline` ⭐⭐⭐ **ВАЖНО!**
- **Предустановка:** `veryfast` (для x264) или `Low Latency` (для NVENC)
- **Настройка:** `zerolatency`

**Кодировщик аудио:**
- **Кодировщик:** `AAC`
- **Битрейт:** `128 Kbps`
- **Частота дискретизации:** `48 kHz`

---

### 3️⃣ **Настройки Видео (Video)**

**Файл → Настройки → Видео**

- **Базовое разрешение:** `1280x720` (ваше разрешение экрана)
- **Выходное разрешение:** `1280x720` ⭐
- **Фильтр масштабирования:** `Ланцош` (лучшее качество для текста)
- **Общие значения FPS:** `25` ⭐

**Почему 720p и 25fps?**
- FFmpeg на сервере автоматически создаст 3 профиля (240p, 360p, 480p)
- 25fps достаточно для текста, меньше нагрузка на сеть
- Для текста не нужно 60fps

---

### 4️⃣ **Расширенные настройки (Advanced)**

**Файл → Настройки → Расширенные**

#### Вкладка "Видео":
- **Цветовой формат:** `NV12`
- **Цветовое пространство:** `709`
- **Диапазон цветов:** `Partial` (Limited)

#### Вкладка "Сеть":
- **Включить отсроченную публикацию:** `Нет`
- **Автоматически переподключаться:** `Да` ⭐
- **Задержка повтора:** `5 секунд`
- **Максимальное количество повторов:** `10`

## 🎨 Рекомендации для текстового контента

### Оптимальные настройки шрифта в источниках:

1. **Источник → Текст (GDI+)**
   - **Шрифт:** Arial, Verdana, или Roboto
   - **Размер:** `48-72` (в зависимости от разрешения)
   - **Стиль:** `Жирный` (Bold) для лучшей читаемости
   - **Цвет:** Контрастный (белый текст на тёмном фоне или наоборот)

2. **Контраст:**
   - Избегайте светло-серого текста на белом фоне
   - Лучшие комбинации:
     - ✅ Белый текст на чёрном фоне
     - ✅ Чёрный текст на белом фоне
     - ✅ Жёлтый текст на синем фоне

## 🎯 Что происходит на сервере?

После того как вы начнёте трансляцию в OBS:

1. **OBS** отправляет поток `1280x720 @ 1500 Kbps` на сервер
2. **NGINX** принимает RTMP поток
3. **FFmpeg** автоматически запускается и создаёт **3 профиля**:
   - 📱 **240p @ 400 Kbps** — для 3G и слабых сетей
   - 📱 **360p @ 600 Kbps** — для 4G
   - 💻 **480p @ 900 Kbps** — для WiFi и десктопов
4. Пользователи получают **adaptive HLS** — их устройство автоматически выбирает лучшее качество

## 🚨 Что делать если проблемы?

### Проблема: "Не удаётся подключиться к серверу"
- Проверьте сервер: `rtmp://spoken-word.ru/live`
- Проверьте ключ: `main`
- Проверьте интернет соединение

### Проблема: "Буферизация / лаги на мобильных"
- Уменьшите битрейт до `1200 Kbps`
- Убедитесь что keyframe interval = `2`

### Проблема: "Плохое качество текста"
- Увеличьте размер шрифта
- Используйте жирный шрифт
- Проверьте контрастность
