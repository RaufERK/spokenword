## ❗ ПРИЧИНА КРАША:

**Next.js загружает файл ТАК:**
```typescript
const bytes = await file.arrayBuffer()  // 509MB в RAM
const buffer = Buffer.from(bytes)       // +509MB копия
await writeFile(tempFilePath, buffer)   // +509MB при записи
// + V8 GC overhead
```
