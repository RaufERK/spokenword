## 💬 6. ЧАТ - КАК В TELEGRAM!

### НОВАЯ СТРУКТУРА:

```
┌──────────────┬────────────────────────────────┐
│              │                                │
│ 📢 Общий чат │  Общий чат                     │
│              │  Общение участников            │
│ 🛟 Поддержка │ ┌────────────────────────────┐ │
│   (2)        │ │                            │ │
│              │ │ Rauf Erik [SUPER]          │ │
│ 💬 Елеонора  │ │ "Добро пожаловать!"        │ │
│              │ │ [👍 12] [❤️ 8] [🔥 5]       │ │
│ 💬 Алексей   │ │                            │ │
│              │ │ Елеонора [USER]            │ │
│              │ │ "Спасибо! Рада быть здесь" │ │
│              │ │ [👍 5] [❤️ 3]              │ │
│              │ │                            │ │
│              │ └────────────────────────────┘ │
│              │                                │
│              │ [Введите сообщение...]         │
│              │ [Ссылка (необязательно)]       │
│              │ [📤 Отправить]                 │
└──────────────┴────────────────────────────────┘
```

### ТРИ ТИПА ЧАТОВ:

**📢 Общий чат:**
- Все видят все сообщения
- Клик на имя → открывается личный чат

**🛟 Поддержка:**
- Пользователи пишут админам
- Админы видят все обращения

**💬 Личные чаты:**
- Один-на-один переписка
- Создаются автоматически при клике на имя

### ФУНКЦИИ:
- ✅ Реакции (👍❤️😊🔥👏)
- ✅ Ссылки в сообщениях
- ✅ Счётчики непрочитанных (красный бейдж)
- ✅ Последнее сообщение в списке
- ✅ Удаление (для админов)
